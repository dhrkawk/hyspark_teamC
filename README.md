# MediCodex - Landing Page Mockup

This is a static landing page for **MediCodex**, a conceptual AI-powered operating system for insurance claims processing and fraud detection.

## Project Overview

This project is a **mockup / prototype** built with Next.js. It does **not** connect to a real backend or database. All data shown (claims, risk scores, reports) is hard-coded for demonstration purposes.

### Key Features Showcased
- **PDF Parsing**: Simulates extracting data from medical claim PDFs.
- **Fraud Detection**: Visualizes risk scoring and anomaly detection.
- **Automated Reporting**: Shows how reports would be generated for adjusters.

## Tech Stack
- **Framework**: Next.js (React)
- **Styling**: Vanilla CSS (CSS Modules / Global Styles)
- **Icons**: Lucide React

## Getting Started

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Run Development Server**:
    ```bash
    npm run dev
    ```

3.  **Open in Browser**:
    Visit [http://localhost:3000](http://localhost:3000) to view the landing page.

## Project Structure
- `pages/index.tsx`: Main landing page.
- `components/`: Reusable UI components (Hero, Features, MockupFlow, etc.).
- `styles/globals.css`: Global design system and utility classes.

## Note
This is a front-end deliverable for demonstration. No real sensitive data is processed.
